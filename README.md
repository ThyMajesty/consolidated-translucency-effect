# Consolidated Window Effects

Unified compositing rules per window type and state for KDE Plasma 6.

## Installation

1. Clone or download this repository.

```bash
git clone https://github.com/thymajesty/consolidated-window-effects.git
```

2. Copy the folder to your KWin effects directory, by default:

```bash
cp -r consolidated-window-effects ~/.local/share/kwin/effects/
```
or
```bash
ln -s /your/full/path/consolidated-window-effects ~/.local/share/kwin/effects/
```

3. Open **System Settings → Window Management → KWin Scripts** and enable **Consolidated Window Effects**.  

4. Configuration available through settings(cog/gear) button.
