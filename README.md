# Consolidated Translucency Effect

Unified translucency rules per window type and state (including fullscreen) for KDE Plasma 6.

## Requirements

KDE Plasma 6.1+

## Installation

1. Clone or download this repository.

```bash
git clone https://github.com/thymajesty/consolidated-translucency-effect.git
```

2. Copy the folder to your KWin effects directory, by default:

```bash
cp -r consolidated-translucency-effect ~/.local/share/kwin/effects/
```
or
```bash
ln -s $(pwd)/consolidated-translucency-effect ~/.local/share/kwin/effects/
```

3. Open **System Settings → Window Management → Desktop Effects** and enable **Consolidated Translucency Effect**.  

4. Configuration available through settings(cog/gear) button.
